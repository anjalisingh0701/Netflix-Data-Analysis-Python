# Netflix Data Analysis using Python

## Project Overview
This project analyzes Netflix Movies and TV Shows data using Python. It focuses on exploratory data analysis and visualization.

## Key Insights
- Movies dominate Netflix's content library.
- Netflix saw massive growth after 2015.
- The United States leads in content production.

## Tools Used
- Python
- Pandas
- Matplotlib
- VS Code

## Dataset
Netflix Movies and TV Shows Dataset from Kaggle.
